## Rock Paper Scissors Game

A simple, interactive Rock Paper Scissors game built with HTML, CSS, and JavaScript where you play against the computer.

### How to Use

1. **Setup**: 
   - Create three files in the same directory:
     - `index.html`
     - `style.css`
     - `script.js`
   - Copy the respective code into each file

2. **Play the Game**:
   - Open `index.html` in your web browser
   - Click on one of the three buttons: Rock (✊), Paper (✋), or Scissors (✌️)
   - The computer will randomly select its choice
   - The result of the round will be displayed immediately
   - Scores are automatically updated

3. **Game Rules**:
   - Rock beats Scissors
   - Scissors beats Paper
   - Paper beats Rock
   - Same choices result in a tie

4. **Features**:
   - Real-time score tracking
   - Visual feedback with emojis
   - Color-coded results (green for win, red for loss, yellow for tie)
   - Reset button to start a new game
   - Responsive design that works on different screen sizes

5. **Controls**:
   - Click any choice button to play a round
   - Click "Reset Game" to reset scores and start over

### File Structure
- `index.html` - Main HTML structure
- `style.css` - Styling and responsive design
- `script.js` - Game logic and interactivity

The game is ready to play immediately after opening the HTML file in a browser - no additional setup required!
